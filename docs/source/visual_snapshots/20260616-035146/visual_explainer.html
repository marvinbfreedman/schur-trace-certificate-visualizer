What We Are Doing

We are studying whether a Weyl/Volterra kernel is positive. The endpoint model has a tiny negative direction. The moving trace Lambda_a(f) detects that direction. If we remove functions with nonzero endpoint trace, the remaining form becomes positive in the finite tests.
